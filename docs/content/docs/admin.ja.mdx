FoxBlaze は PM2 を用いて Linux サーバー上で効率的に実行できるように設計されています。

## 前提条件

- **Node.js** (v20+)
- **PostgreSQL**
- **Redis**
- **PM2**

## アプリの設定と環境構築

```bash
git clone https://github.com/noweieth/foxblaze-trade.git
cd foxblaze-trade
npm install
npx prisma generate
npx prisma db push
```

ルートに `.env` ファイルを作成し、必要な設定値を追加してください。(BIP-39 マスターシードなど)。

## 起動方法

```bash
npm run build
pm2 start dist/main.js --name foxblaze
pm2 save
```
