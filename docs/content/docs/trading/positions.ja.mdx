## `/positions` コマンド

`/positions` を送信すると、美しくデザインされた **PNL カード** がダイナミックに生成されます。これには、未実現 PNL、ROE、チャート情報などが記載されています。

## ポジションの決済 (クローズ)

パネルから `✖️ CLOSE` をクリックするか、または `/close BTC` や `/close ALL` コマンドを使用していつでも簡単にポジションを決済できます。決済後は確定した PNL 画像が送信されます。
