FoxBlaze lets you trade 100+ perpetual futures assets on Hyperliquid with zero gas fees, directly from Telegram.

## Commands

| Command | What it does |
|---|---|
| `/long` | Open a LONG position |
| `/short` | Open a SHORT position |
| `/positions` | View your active trades with live PnL |
| `/close BTC` | Close a specific position |
| `/close all` | Close all positions at once |
| `/orders` | View pending orders |
| `/history` | See your recent trade history |
| `/chart BTC` | Interactive candlestick chart |
| `/price BTC` | Quick price check |
| `/balance` | Check your account balance |
