FoxBlaze는 NestJS 프레임워크에 동적으로 구축되었으며 PM2를 사용하여 Linux 서버에서 효율적으로 실행되도록 설계되었습니다.

## 시스템 요구 사항

봇을 배포하기 전에 서버에 다음 시스템 종속성이 설치되어 있는지 확인하십시오.
- **Node.js** (v20 이상 권장)
- **PostgreSQL** (Prisma 지갑 및 상태 지속성에 필요)
- **Redis** (BullMQ 비동기 작업 실행에 필요)
- **PM2** (백그라운드 프로세스 관리용)

## 애플리케이션 설정

1. **저장소 복제** 및 Node 종속성 설치:
```bash
git clone https://github.com/noweieth/foxblaze-trade.git
cd foxblaze-trade
npm install
```

2. **데이터베이스 생성**: Prisma 스키마를 초기화하고 트랜잭션을 적용합니다.
```bash
npx prisma generate
npx prisma db push
```

## 환경 구성

루트 디렉토리에 `.env` 파일을 만듭니다.

```bash
TELEGRAM_BOT_TOKEN=your_bot_token_from_botfather
DATABASE_URL="postgresql://user:pass@localhost:5432/foxblaze?schema=public"
REDIS_HOST=127.0.0.1
REDIS_PORT=6379
MASTER_SEED_PHRASE="your twelve words secret phrase goes here"
AES_SECRET_KEY="32_byte_aes_encryption_key_here"
ARBITRUM_RPC_URL="https://arb1.arbitrum.io/rpc"
```

<Callout title="보안 주의" type="error">
`MASTER_SEED_PHRASE` 및 `AES_SECRET_KEY`는 최고 수준의 보안으로 취급되어야 합니다. 유출 시 전체 자금이 위험할 수 있습니다.
</Callout>

## 배포 (PM2)

```bash
npm run build
pm2 start dist/main.js --name foxblaze
pm2 save
```
