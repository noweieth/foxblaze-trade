FoxBlaze автоматически переводит USDC из сети Arbitrum в Hyperliquid через L1 сканирование.

## Как внести депозит

1. **Скопируйте адрес**: Используйте `/start`, чтобы получить свой адрес.
2. **Отправьте USDC**: Отправьте нативный USDC в сети **Arbitrum One**. Минимум 10 USDC.
3. **Сканирование**: Бот непрерывно мониторит ваш адрес с помощью `DepositService`.

Как только баланс превысит 1.0 USDC, будет запущена очередь `AUTO_DEPOSIT`.

<Callout title="Почему Arbitrum?" type="info">
Нативный мост L1 работает через Arbitrum, позволяя выполнять быстрые переводы на биржу без токена газа EVM (без ETH).
</Callout>
