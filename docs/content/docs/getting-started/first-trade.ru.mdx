После депозита вы можете начать торговать с помощью FoxBlaze.

## Команды

1. Отправьте `/long` или `/short`.
2. Бот попросит тикер актива (например, `BTC`).
3. Появится **Интерактивная панель настройки ордера**.

```text
🦊 FOXBLAZE ORDER SETUP 🦊

🪙 Asset: BTC
📊 Side: LONG
💰 Margin: $10
🚀 Leverage: 10x
🎯 TP Price: None
🛡️ SL Price: None
```

## Настройка

Вы можете настроить параметры:
- **Size (Размер)**: Измените маржу позиции.
- **Leverage (Плечо)**: Измените кредитное плечо.
- **TP / SL**: Установите триггеры фиксации прибыли или стоп-лосс.

## Подтверждение

Нажмите **Confirm Trade**, бот проверит риск-лимиты и разместит маркет-ордер на Hyperliquid.
