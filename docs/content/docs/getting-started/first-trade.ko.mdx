---
title: 첫 거래
description: 무기한 선물 거래 실행하기.
---

입금 후 바로 거래를 시작할 수 있습니다.

## 명령

1. `/long` 또는 `/short`를 보냅니다.
2. 자산 티커(예: `BTC`)를 입력합니다.
3. **대화형 설정 패널**이 나타납니다.

```text
🦊 FOXBLAZE ORDER SETUP 🦊

🪙 Asset: BTC
📊 Side: LONG
💰 Margin: $10
🚀 Leverage: 10x
🎯 TP Price: None
🛡️ SL Price: None
```

## 설정 패널

인라인 키보드를 사용하여 매개 변수를 조정할 수 있습니다. 패널을 통해 크기, 레버리지, 이익 실현/손절매를 설정합니다.

## 확인

준비가 되면 **Confirm Trade**를 클릭합니다. 봇이 Hyperliquid 거래소에 시장가 주문을 넣습니다.
