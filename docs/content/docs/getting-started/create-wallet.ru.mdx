---
title: Создание кошелька
description: Как сгенерировать некастодиальный торговый кошелек.
---

Для начала, начните диалог с Telegram-ботом FoxBlaze и введите `/start`.

## Пошаговое руководство

1. **Инициализация**: Бот распознает ваш Telegram ID и сгенерирует безопасный **HD Wallet (BIP-39/44)** в сети Arbitrum.
2. **Приватные ключи**: Ваши ключи генерируются безопасно и шифруются с помощью AES-256. Мы никогда не имеем доступа к вашим средствам.
3. **Адрес кошелька**: Бот ответит вашим новым адресом для депозита.

```text
✅ Registration Successful!

💳 Your Trading Wallet (Arbitrum)
0x1234abcd5678...
```

<Callout title="Важно" type="info">
Для маржинальной торговли требуется минимальный первый депозит в размере 10 USDC в сети Arbitrum для авторизации Agent API на Hyperliquid.
</Callout>
