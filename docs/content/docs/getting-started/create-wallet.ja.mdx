---
title: ウォレットの作成
description: 非カストディアルな取引ウォレットを生成します。
---

Telegram ボットに対し `/start` と入力してください。

1. **初期化**: ボットが Telegram ID を認識し **HD ウォレット** を Arbitrum ネットワーク上に生成します。
2. **プライベートキー**: 自動で安全に暗号化されます。
3. **ウォレットアドレス**: デポジット用のアドレスが表示されます。

<Callout title="注意" type="info">
マージン取引のロックを解除するには、Agent API 承認の要件として Arbitrum 上に最低 10 USDC の入金が必要です。
</Callout>
